# capstone2026

There is a gap in accurate, consistent, easy to find, and detailed information about public venue accessibility online.  Users are not able to find reliable accessibility information online that would help them plan and prepare for visits.

We are building an accessibility feedback/reviewing platform for public venues in the city of Seattle. The platform will provide detailed accessibility features for different locations and allow people with disabilities, older adults, caregivers, and other community members to share lived experiences about venue accessibility.
